📚 Student Productivity Suite

A web-based Student Productivity Suite designed to help students organize their studies, manage time effectively, and improve overall learning efficiency.

---

## 🚀 Features

- 📝 **Notes Module**  
  Create, save, and manage study notes easily.

- ❓ **Quiz Module**  
  Practice multiple-choice questions and test your knowledge.

- 📅 **Timetable Module**  
  Generate and manage a weekly study schedule.

- ⏳ **Pomodoro Timer**  
  Boost focus with timed study sessions and breaks.

- 🧠 **Flashcards Module**  
  Create and review flashcards for quick revision.

---

## 🛠️ Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Logic and interactivity  
- **LocalStorage** – Data persistence in browser  

---

## 💡 How It Works

1. Open the application in your browser  
2. Choose a module (Notes, Quiz, Timetable, etc.)  
3. Add and manage your study data  
4. All data is automatically saved using LocalStorage
